## 启动服务

镜像启动后，您可以通过浏览器访问以下地址登录 Halo：

```text
服务访问地址: http://服务器IP
```

首次访问会进入 Halo 初始化页面，填写初始化信息即可完成 Halo 的安装。

## 开放端口

为了确保外部能够正常访问 Halo 服务，您需要在阿里云服务器的安全组规则中开放 `80` 端口。

具体的开放步骤可以参考阿里云的 [端口放行教程](https://help.aliyun.com/document_detail/25471.html)。
