## 前提条件

在开始使用商城功能之前，我们需要先确定以下条件是否已经满足：

- 已经正常安装了 [Halo 商业版](../../getting-started/install/docker-compose.md)
- 已经激活了 [Halo 商业版许可证](../../user-guide/activate.md)
- 根据当地法律法规，满足开设网络商店的资质与要求
- 开通支付平台账户，目前支持：
  - [支付宝官方接口](https://open.alipay.com/)
  - [微信支付官方接口](https://pay.weixin.qq.com/)

## 前台路由

目前商城包含以下前台路由：

- `/shop`：商城首页，目前会重定向到商品列表页面
- `/shop/products`：商品列表页面
- `/shop/products/:id`：商品详情页面
- `/shop/cart`：购物车页面
- `/shop/checkout/:id`：结算页面
- `/shop/order/:code/payments`：支付页面

:::note
这些页面目前无需主题提供对应的模板文件也可以访问，使用的是系统内置的模板文件，如果你当前激活的主题提供了对应的模板文件，那么会优先使用主题提供的模板文件。
:::

你可以按照需求，将这些页面路径添加到[网站菜单](../menus.md)中。
